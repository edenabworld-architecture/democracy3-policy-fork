# Democracy 3.0 v1.0 운영 지침

## 자동 순환

매일: 국회수집 → 시민 공개로그 동기화 → 정책 버전·시행추적 생성 → 무결성검사 → 공개배포.

## 시민 제안 처리

접수 → 검토 중 → 근거 확인 → 반영/부분반영/기각/보류. 결과에는 결정자·일자·이유·반영 버전을 남깁니다. 원문은 삭제하지 않습니다.

## 시행 책임추적

시범 10건은 `review_overrides.json`의 `implementation_tracking`에 공식 상태, 실행률, 계획·집행예산, 지표, 문제, 장점, 출처를 기록합니다. 자동 추적기는 이를 `policy-lifecycle.json`에 병합합니다.

## 전국 확장

`source_registry.json`의 공식 출처별 어댑터를 연결합니다. 현재 국회 법안만 active이며, 법령 API는 인증값, 지방·재정 자료는 정책 ID와 지역·사업코드 매핑이 필요합니다.
