# 프로젝트 헌장 v0.1

## 목적
국회·정부 정책을 일반 국민이 이해할 수 있는 언어로 설명하고, 목적·비용·권한·현실성·부작용·대안을 검토한다. 시민과 전문가가 수정안과 대체안을 포크로 제안하며, 처리와 시행 결과까지 추적한다.

## 금지
- 근거 없는 정당·의원 서열화
- 정치성향 자동추정
- AI 점수를 객관적 진실로 표현
- 출처 없는 단정
- 자동 생성 결과의 무검수 공식화
- 시민 개인정보 판매
- 분석결과 은폐

## 검토 상태
자동수집 / 규칙검사 / AI 초안 / 시민 검토 / 전문가 검토 / 정정됨 / 시행결과 추적
