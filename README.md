# Democracy 3.0 정책포크

국회와 정부의 정책을 누구나 이해할 수 있는 언어로 풀고, 주장·근거·위험·악용경로·대체조문과 시행결과를 공개적으로 검증하는 정책 버전관리 플랫폼입니다.

**창안 및 원안 설계: 박찬성**

## 현재 구현

- 제22대 국회 최신 법안 자동수집
- 공식 제안이유 및 주요내용 연동
- 국민용 설명과 복수 위험프로필
- 주장 원장·기각조건·레드팀 공격경로
- A/B/C 정책포크 설계초안
- 국가 시범검토 10건 고정 운영
- 공식문서와 공통 내비게이션 링크 분리
- 구조화·공식자료·독립근거·사람검토 성숙도 분리
- 검증된 인간 작업을 `review_overrides.json`에 영구 보존
- 자동생성 데이터와 수동검증 데이터를 매 실행마다 병합
- 공개 전 통합 무결성 검사
- 오류·반론·정정·검토 이력 공개

## 파일 구조

```text
fetch_bills.py                국회 API 수집과 자동분석
pilot_research.py             국가 시범검토 선정·공식자료 조사
pilot_manifest.json           고정된 시범검토 10건
review_overrides.json         사람이 검증한 자료와 조문 원장
review_store.py               수동검증 원장 검증·병합
validate_release.py           공개 전 통합감사
pilot-editor.html             운영자용 JSON 편집기
probe_official_documents.py   국회 첨부문서 접근경로 진단
bills.json                    하위호환 전체 데이터
bills-index.json              경량 목록
reports/                      법안별 상세보고서
release-status.json           배포검사 결과
```

## 운영 원칙

1. 공식자료·발의자 주장·자동추론·사람검토를 구분합니다.
2. 근거 없는 찬반 결론을 자동으로 확정하지 않습니다.
3. 시범검토 10건을 새 법안 때문에 임의 교체하지 않습니다.
4. 사람이 검증한 결과는 자동생성 파일이 아니라 별도 원장에 기록합니다.
5. 오류와 반론을 삭제하지 않고 정정이력으로 보존합니다.
6. 실제 조문과 출처 없이 조문 대조나 포크 완료로 표시하지 않습니다.

## 운영자 검토

1. `pilot-editor.html`을 엽니다.
2. 공식문서·독립근거·조문대조·대체조문·검토기록을 입력합니다.
3. `review_overrides.json`을 내려받습니다.
4. 저장소의 같은 파일을 덮어씁니다.
5. Actions의 `Update and validate policy data`를 새로 실행합니다.

엄격한 형식검사는 `validate_release.py`가 수행합니다. 잘못된 확정판정, 출처 없는 조문, 검토자 없는 완료표시가 있으면 배포를 중단합니다.

## 한계

현재 자동검토는 공식 요약 기반의 시작점입니다. 법률안 전문, 신·구조문, 비용추계, 위원회 자료, 독립 통계와 사람 검토가 채워져야 완성보고서가 됩니다. 본 사이트는 법률자문이나 국가기관의 공식 판단을 대체하지 않습니다.
