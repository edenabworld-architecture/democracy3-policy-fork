# Democracy 3.0 정책포크 v1.0

국가부터 읍·면·동·리까지 모든 정책을 **30초 이해 → 주장·근거·논리 분리 → 적대적 레드팀 → 시민 주석·포크 → 공개 결정·버전 → 시행·예산·성과 추적**의 동일한 구조로 관리하는 공공 정책 인프라입니다.

**창안 및 원안 설계: 박찬성**

## v1.0 핵심

- 30초 이해 6칸: 문제·변경·영향대상·발의자 기대·최대위험·판단질문
- 정치인의 주장·제시근거·논리사슬 분리
- 권한집중·특혜·부패·감시·처벌·예산누수 적대검증
- 시민 주석·정정·근거·레드팀·포크 작성 화면
- 공개 접수·처리상태와 시민 로그 자동 동기화
- 정책 상태 변경의 자동 버전 생성
- 통과 후 실행률·예산·성과·피해·재검토 추적
- 국가·시도·시군구·읍면동리 정책지도와 공식 데이터 연결목록
- 매일 자동수집·시민로그·정책생애주기·무결성검사·배포

## 정직한 현재 범위

국회 최신 법안 수집은 실제 가동 중입니다. 중앙정부 입법예고·현행법령·자치법규·지방재정은 공식 출처와 표준 데이터 구조를 등록했지만, 각 수집 어댑터와 인증키는 순차 연결이 필요합니다. 익명 시민접수는 별도 서버를 배포해야 하며, 그 전에는 사이트 내 초안·JSON 내려받기와 GitHub 공개접수를 사용합니다.
