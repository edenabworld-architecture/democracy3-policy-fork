# Democracy 3.0 정책포크 v0.1

대한민국 국회 법률안을 자동 수집하고, 국민용 설명·쟁점·현실성·정책 포크를 공개하는 무료 플랫폼의 시작 파일입니다.

## 원칙
- AI는 1차 분석자이며 최종 심판자가 아닙니다.
- 정당·의원보다 정책 구조를 평가합니다.
- 분석근거·오류·수정이력을 공개합니다.
- 국민은 회원가입 없이 무료로 열람합니다.
- 자동분석과 사람 검토 상태를 구분합니다.

## 시작
1. GitHub 공개 저장소 `democracy3-policy-fork`를 만듭니다.
2. 이 압축파일의 내용을 업로드합니다.
3. 열린국회정보 Open API 인증키와 서비스명을 GitHub Secrets에 등록합니다.
4. GitHub Pages로 `site` 폴더를 배포합니다.
5. 샘플 데이터를 실제 의안 데이터로 교체합니다.

샘플 정책은 모두 가상 자료입니다.
